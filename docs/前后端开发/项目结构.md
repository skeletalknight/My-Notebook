# 常见项目结构

## 前端文件目录

| 目录       | 描述                           |
| ---------- | ------------------------------ |
| assets     | 所有静态资源文件               |
| components | 复用的组件                     |
| hooks      | 自定义的hook函数               |
| routes     | 有关路由的文件夹               |
| types      | 复用的类型/接口定义            |
| utils      | 自定义的纯函数                 |
| pages      | 页面文件夹-对应有api/组件/网页 |
| config     | 全局常量                       |

## 辅助开发插件

- Eslint：	代码格式检查

- Husky： git提交自动化工作流

- Jest：  集成测试

- css预处理器：

	- less
	- scss
	- tailwind

- 状态管理：

	- redux

	- mobx

- Drizzle： 更安全的数据库调用
